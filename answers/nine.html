1071. Greatest Common Divisor of Strings

For two strings s and t, we say "t divides s" if and only if s = t + ... + t (i.e., t is concatenated with itself one or more times). Given two strings str1 and str2, return the largest string x such that x divides both str1 and str2.

Example : Input: str1 = "ABABAB", str2 = "ABAB" Output: "AB"

https://leetcode.com/problems/greatest-common-divisor-of-strings/description/


        class Solution {
    public String gcdOfStrings(String str1, String str2) {
        int n = str1.length();
        int m = str2.length();

        if(n < m)
            return gcdOfStrings(str2,str1);
        else if(!str1.startsWith(str2))
            return "";
        else if(str2.length()==0) 
            return str1;
        else{
            return gcdOfStrings(str1.substring(str2.length()),str2);  //remove the start of the larger string which matches with shorter string
        }
    }
}
